# Diagramas C4 em Structurizr DSL

Versão alternativa dos diagramas C4 do relatório, em DSL do Structurizr.

## Como visualizar

Abrir `https://structurizr.com/dsl` no navegador e colar o conteúdo de `workspace.dsl` no editor. As três views (Contexto, Conteineres, Componentes) ficam disponíveis pra navegação e exportação.
